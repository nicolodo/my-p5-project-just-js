How It Works

A basic p5.js sketch uses two main functions:

setup() – Runs once when the program starts

draw() – Runs continuously to create animation


function setup() {
    createCanvas(400, 400);
}

function draw() {
    background(220);
    circle(200, 200, 100);
}
